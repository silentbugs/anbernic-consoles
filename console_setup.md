## 1. Todo

First you need to choose a firmware for your console. A common one for RG35XXSP and RG40XXV is KNULLI.

This guide assumes you have already flashed the ROM of your choice.

### 1.1 Initial
- disable home music
	sound settings -> music -> frontend music (off)
- connect to wifi
- update system
	updates & downloads -> software updates -> start update
- set timezone
	system -> timezone -> Europe/Athens
- change launch transition
	User interface -> Game Launch Transition -> Fade
- scraper
	- create credentials & login
	- image source -> Box 2D
	- Scrape For -> Community rating, Video
- ONLY RG35XXSP:
	Settings -> Power Management -> Lid Close Mode -> Shutdown

### 1.2 over usb
- add roms
	- find rom directory
- add bios
	- add to /bios/
- update logo.png (splash screen)

### 1.3 Final

- change screenscraper login

### 1.4. Extra

- autosave
	Game Settings -> Save States -> Auto Save/Load
- Quick resume mode (launch last playing game after boot from shutdown)
	Game Settings -> Save States -> Quick Resume Mode

# 2. User info:

- Connect to WiFi
- Check for updates
	updates & downloads -> software updates -> start update
- Add game to favorites: Hold X
- Exit game:
	Menu & Start button
- Turn d-pad to analog stick: click power button once (do not hold)
- Brightness: Menu + Vol Up/Down
